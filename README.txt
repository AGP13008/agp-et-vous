AGP & Vous – Mode d'emploi

1. Créez gratuitement un compte GitHub.
2. Créez un nouveau dépôt public, par exemple : agp-app.
3. Déposez tous les fichiers de ce dossier dans le dépôt.
4. Dans Settings > Pages, choisissez "Deploy from a branch", puis la branche main et le dossier root.
5. GitHub vous donnera une adresse gratuite.
6. Ouvrez cette adresse sur votre téléphone et ajoutez-la à l'écran d'accueil.

Fichiers à modifier :
- index.html : textes, dates, numéros et liens.
- styles.css : apparence.
- app.js : fonctionnement des boutons.

Important :
L'adresse association.agp@example.com est fictive. Remplacez-la par l'adresse réelle de l'association.
